# ⚽ Football xG Analyst

An interactive football analytics platform that leverages **Machine Learning, Historical Football Data, and Explainable AI** to estimate the probability of a shot resulting in a goal (**Expected Goals - xG**). Built using **StatsBomb Open Data** and deployed through **Streamlit**, the application allows users to explore shot quality, compare historical scenarios, and gain intuitive tactical insights through an interactive dashboard.

---

## 📖 Overview

Expected Goals (**xG**) has become one of the most widely used metrics in modern football analytics. Instead of evaluating shots solely based on whether they resulted in a goal, xG estimates the probability of scoring by considering contextual factors such as shot location, technique, body part, and game situations.

Football xG Analyst aims to make football analytics more accessible to both:

- 📊 Football Analysts and Data Scientists
- ⚽ General Football Fans and Enthusiasts
- 🎓 Students interested in Sports Analytics and Machine Learning

The application not only predicts xG values but also explains them by retrieving historically similar shots and generating contextual insights.

---

## 🚀 Features

### ⚽ Expected Goals Prediction
Predicts the probability of a shot becoming a goal using a supervised Machine Learning model.

### 🔥 Historical Shot Density Heatmap
Visualizes historically similar shots on a football pitch using kernel density estimation.

### 📊 Similar Historical Shot Retrieval
Uses Nearest Neighbors retrieval to find shots with similar locations and characteristics from historical football data.

### 🧠 Explainable Football Analytics
Automatically generates contextual explanations regarding the quality of a scoring opportunity.

### 🎯 User-Friendly Interface
Supports:

- Pitch Zones for casual users
- Advanced Coordinates for analysts

### 🌙 Dark-Themed Dashboard
Interactive and visually appealing dashboard designed for an enhanced user experience.

# 🤖 Machine Learning Pipeline

## Model
**Logistic Regression**

Logistic Regression was selected because:

- Interpretable probabilities
- Computational efficiency
- Strong baseline performance
- Suitable for probability estimation problems

The model outputs:

```
P(Goal | Shot Characteristics)
```

which is interpreted as:

```
Expected Goals (xG)
```

---

# 📈 Historical Shot Retrieval

A Nearest Neighbors retrieval system was developed to identify historically similar shots.

Inputs:

- Shot location
- Contextual shot features

Outputs:

- Similar historical shots
- Historical outcomes
- Historical xG values
- Players and teams involved

This component enables users to compare hypothetical scenarios against real football events.

---

# 🧠 Explainability Layer

The explainability engine automatically categorizes chances into:

- Low Quality Chance
- Decent Opportunity
- High Quality Chance
- Clear-Cut Opportunity

The system then generates human-readable insights explaining why a particular shot has a certain probability of resulting in a goal.
